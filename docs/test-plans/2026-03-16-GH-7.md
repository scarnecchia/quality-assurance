# Human Test Plan: DuckDB Global Checks Migration (GH-7)

## Prerequisites

- Python 3.12+ environment with `uv` installed
- `uv run pytest` passing (416 tests green)
- Access to at least one Parquet file and one SAS7BDAT file in SCDM format
- DuckDB available (included in project deps, >= 1.2.0)
